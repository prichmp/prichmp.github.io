Write Forward

The Write Forward framework is a logging library written in Java. It is designed to help developers write easy-to-use, portable, and future-proof logging code. Write Forward is a layer that sits on top of other logging frameworks - such as Logback, Log4j2, and java.util.logging. Write Forward by itself has no ability to write the log messages to a file. Instead Write Forward exposes an API and then forwards the logging message to your choice of logging framework. If the logging framework changes in your project at some future point no changes to your code is required.

Features

  • Use with your favorite logger framework including Log4j2, Logback, java.util.logging, and SLF4J.
  • Log interesting variables in your code easily and quickly.
  • Output to multiple loggers at the same time.
  • Has a "fast" and "slow" setting for production and development respectively.
  • See what line number the logger was called from in "slow" mode.
  • Automatically checks the underlying framework to see if the level is enabled.
  • The framework is extensible, with the ability to implement logging to your own database or other location.
  • Use a fluent syntax to compose logging messages that get formatted and output.
